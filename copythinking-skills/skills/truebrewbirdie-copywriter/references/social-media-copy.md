# Social Media Copy

Write engaging posts for Twitter/X, LinkedIn, Instagram, and threads that build audience and drive action.

---

## Core Principle

> "Social media is a machine gun. One post won't change everything, but consistent posts will."

The goal: Build trust, demonstrate expertise, create demand.


### Engagement Boosters

- Add images/screenshots to key points
- End with a question or retweet request
- Quote-tweet your opener with a hook

## Carousel Content Structure

### Educational Carousel (7-10 slides)
```
Slide 1: Hook title (problem or promise)
Slide 2: Context/Why this matters
Slides 3-8: One point per slide
Slide 9: Summary/Key takeaway
Slide 10: CTA + Follow prompt
```

```

```

## Platform-Specific Best Practices

### Twitter/X
- Hook in first line (no fluff)
- 1-2 tweets optimal for engagement
- Threads for deep content
- Use line breaks generously
- Add value before asking for anything

### LinkedIn
- Personal stories outperform tips
- First 2 lines show in preview - make them count
- Use emojis sparingly
- Post in first person
- 1,300+ characters perform well
- Engage in comments quickly

### Instagram
- First line is the hook (shows in feed)
- Line breaks for readability
- Use "..." to encourage "more" click
- Hashtags in caption, not comments
- CTA at the end
- Carousel > Single image for reach

## Content Pillars for Consistency

### Build a 4-Pillar System:
1. **Educational** - Teach something valuable
2. **Inspirational** - Motivate or share wins
3. **Personal** - Behind-the-scenes, stories
4. **Promotional** - Offers, products, services (20% max)

### Content Ratio:
- 40% Educational
- 20% Inspirational
- 20% Personal/Story
- 20% Promotional

## Quick Checklist

- [ ] Does the first line stop the scroll?
- [ ] Is it written for ONE specific reader?
- [ ] Is there a clear value proposition?
- [ ] Is it easy to skim (short paragraphs, bullets)?
- [ ] Does it end with engagement trigger?
- [ ] Is it platform-appropriate in tone and length?
- [ ] Would YOU engage with this post?
