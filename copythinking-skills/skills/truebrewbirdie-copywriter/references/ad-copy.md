# Ad Copy Templates

Write scroll-stopping ads for Facebook, Instagram, Google, LinkedIn, and YouTube.

---

## Core Principle

> "You have 3 seconds. Hook or die."

Ads interrupt. They must earn attention immediately and deliver value fast.

---

## Universal Ad Structure

```
1. HOOK (Stop the scroll - 1 line)
2. PROBLEM (Relate to their pain - 1-2 lines)
3. SOLUTION (Your offer - 1-2 lines)
4. PROOF (Credibility element - 1 line)
5. CTA (Clear action - 1 line)
```

---

## Hook Formulas (The First Line)

### Pattern Interrupt Hooks
- "Stop scrolling if you [specific situation]"
- "Unpopular opinion: [contrarian take]"
- "The worst advice I ever got about [topic]"

### Curiosity Hooks
- "I spent $50,000 to learn this..."
- "What if [unexpected possibility]?"
- "Nobody talks about this, but..."

### Direct Benefit Hooks
- "[Result] in [Timeframe] (without [pain point])"
- "How I [achieved result] with [unusual method]"
- "The [number] [things] every [audience] needs to know"

### Question Hooks
- "Ever wonder why [frustrating situation]?"
- "What would you do with [desirable outcome]?"
- "Tired of [common problem]?"

### Story Hooks
- "Last Tuesday, I almost [dramatic moment]..."
- "My client called me at 2am because..."
- "3 years ago, I was [relatable bad situation]..."

## Google Ads Templates

### Search Ad (Character Limits: 30/30/30 headlines, 90/90 descriptions)

**Headlines (30 chars each):**
```
H1: [Keyword] + [Benefit]
H2: [Social Proof] or [Urgency]
H3: [CTA] or [Differentiator]
```

**Descriptions (90 chars each):**
```
D1: [Problem statement]. [Solution intro]. [Key benefit].
D2: [Proof point]. [Urgency]. [CTA with offer].
```

**Example:**
```
H1: Email Marketing That Sells
H2: 2,000+ Businesses Trust Us
H3: Free 14-Day Trial

D1: Tired of emails that get ignored? Send campaigns that convert. 3x avg open rates.
D2: Join 2,847 businesses sending smarter emails. Start free today. No card needed.
```

### Display Ad Copy

**Short (25 chars):**
```
[Benefit] + [Timeframe]
```

**Medium (35 chars):**
```
[Problem]? [Solution] Today
```

**Long (90 chars):**
```
[Specific audience]: [Problem statement]. [Solution] helps you [result]. [CTA].
```

## YouTube Ad Scripts

### 5-Second Hook Formula

```
"If you [situation], watch this because [promise]"
```

### 15-Second Bumper

```
[0-3s] HOOK: [Pattern interrupt or question]
[4-10s] PROMISE: [What they'll get]
[11-15s] CTA: [What to do next]
```

### 30-60 Second Ad

```
[0-5s] HOOK
Stop if you're [specific audience].

[6-20s] PROBLEM + AGITATE
You've probably tried [common solution].
But [why it doesn't work].
And it's costing you [consequence].

[21-45s] SOLUTION + PROOF
[Product] is different because [mechanism].
[Social proof or result].

[46-60s] CTA + URGENCY
Click below to [action].
[Bonus/offer].
[Urgency element].
```

## Ad Copy Checklist

- [ ] Does the hook stop the scroll in 1 line?
- [ ] Is the problem/pain clear and relatable?
- [ ] Is the solution specific, not vague?
- [ ] Is there a proof element (social, numbers, testimonial)?
- [ ] Is the CTA crystal clear (one action)?
- [ ] Is it written for ONE specific audience?
- [ ] Does it match the awareness level of the audience?
- [ ] Is it platform-appropriate (length, tone, format)?

---

## Platform-Specific Notes

### Facebook/Instagram
- Use emojis sparingly but strategically
- Short paragraphs (1-2 lines)
- Include line breaks for readability
- Videos: First 3 seconds critical
- Carousel: Hook on card 1

### LinkedIn
- Professional but not corporate
- Lead with insight, not pitch
- Longer copy performs well
- Avoid emojis in B2B

### Google Search
- Include keywords naturally
- Focus on search intent
- Use numbers and specificity
- Strong CTA in description

### YouTube
- Script the first 5 seconds carefully
- Speak to camera for trust
- Clear visual CTA
- Test multiple hooks
