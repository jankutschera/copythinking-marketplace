# AEO Best Practices Reference

## Citation-Worthy Content Structure

### The "Answer First" Pattern
Place direct answers in first 100 words:

```
Question: How much does X cost?

GOOD:
X costs $50-200 depending on size. Basic models start at $50, 
professional versions cost $150-200. This includes shipping and setup.

[Then expand with details...]

BAD:
There are many factors that affect pricing. Let's explore the 
history of X and understand market dynamics...
[Answer buried on line 47]
```

### Question-Answer Pairs
Structure content as Q&A:

```markdown
## What is [Topic]?
[Direct 2-3 sentence answer]

[Optional: Expanded explanation]

## How does [Topic] work?
[Direct answer with steps]

1. First step
2. Second step
3. Third step

## When should I use [Topic]?
[Direct answer with scenarios]
```

### The Layered Answer Pattern
Build answers in layers for different depths:

```markdown
## Quick Answer
[2-3 sentences - standalone, complete answer]

## Detailed Explanation
[2-3 paragraphs with context and nuance]

## Technical Details
[Deep dive for experts]

## Common Questions
- **Q: [Related question]**
  A: [Direct answer]
```

## Semantic Optimization Techniques

### Entity Definition Pattern
Define entities clearly for AI understanding:

```markdown
[Entity Name] is a [category] that [primary function]. 

Example:
Mailchimp is an email marketing platform that helps businesses 
send newsletters and automated campaigns to subscribers.

Not just:
Mailchimp is a popular tool for emails.
```

### Relationship Mapping
Show how concepts connect:

```markdown
[Tool A] works with [Tool B] to [outcome].
[Tool A] is an alternative to [Tool C].
[Tool A] is required for [Process X].

Example:
Stripe works with Shopify to process customer payments.
Stripe is an alternative to PayPal.
Stripe is required for subscription billing.
```

### Comparison Tables
Use tables for "vs" queries:

```markdown
| Feature | Option A | Option B |
|---------|----------|----------|
| Price | $50/mo | $100/mo |
| Use case | Small teams | Enterprise |
| Setup time | 5 minutes | 1 hour |
```

## Authority Signals

### Data Citation Format
Always cite data sources:

```markdown
GOOD:
According to [Source], 73% of users prefer X over Y (2024 study).

MEDIOCRE:
Studies show most users prefer X.

BAD:
Everyone prefers X.
```

### Expertise Indicators
Show credibility:

```markdown
- Author credentials in byline
- "Based on our analysis of 10,000+ cases..."
- "In our 15 years of experience..."
- "According to research published in [Journal]..."
```

### Update Signals
Show content freshness:

```markdown
**Last updated:** November 2025
**Data current as of:** Q4 2025
**Reviewed:** Quarterly by [Expert Name]
```

## Voice Search Optimization

### Conversational Query Patterns
Write for how people speak:

```markdown
GOOD:
"How much does it cost to install solar panels?"
You can expect to pay $15,000-25,000 for a typical home installation.

BAD:
"Solar Panel Installation Costs"
Installation pricing varies based on multiple factors.
```

### Complete Sentence Answers
Answer in full sentences:

```markdown
GOOD:
Q: Is X worth it?
A: Yes, X is worth it if you need Y or Z. However, if you only 
need basic features, consider the free alternative.

BAD:
Q: Is X worth it?
A: Worth it for Y and Z use cases. Free alternative available.
```

## Content Freshness Strategies

### Update Frequency by Content Type

| Content Type | Update Frequency | What to Update |
|--------------|------------------|----------------|
| News/Trends | Weekly | Stats, examples, current events |
| How-to guides | Quarterly | Screenshots, tool versions, steps |
| Product reviews | When product updates | Features, pricing, comparisons |
| Statistical content | Annually | Data, research citations |
| Evergreen guides | Bi-annually | Examples, best practices |

### Freshness Signals to Include

```markdown
**Published:** November 7, 2025
**Last Updated:** November 7, 2025
**Next Review:** February 2026

**What's new in this update:**
- Updated pricing for Q4 2025
- Added comparison with [New Competitor]
- Refreshed screenshots to current UI
```
