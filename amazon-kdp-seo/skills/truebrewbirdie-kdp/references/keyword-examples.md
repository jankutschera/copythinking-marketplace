## Keyword-Kombinations-Techniken

### Technik 1: Leser-Benefit-Focus
Statt: `fantasy buch roman`
Besser: `fantasy flucht alltag eskapismus abenteuer`

### Technik 2: Vergleichs-Keywords
`fans von [bekannter Autor]`
`ähnlich wie [Bestseller-Titel]`
`für leser von [Genre-Star]`

### Technik 3: Emotionale Keywords
- Gefühle: herzzerreißend, mitreißend, fesselnd, emotional
- Zustände: entspannend, inspirierend, motivierend
- Ergebnisse: lebensverändernd, augenöffnend, transformativ

### Technik 4: Problem-Lösung (Sachbücher)
```
[Problem] + [Lösung/Methode] + [Zielgruppe]
Beispiel: "prokrastination überwinden studenten methoden"
```

## Verbotene Keywords (vermeiden!)

- Subjektive Claims: "bester", "beliebtester", "top", "#1"
- Zeit-sensitiv: "neu", "aktuell", "2024", "2025"
- Generisch: "buch", "kindle", "ebook"
- Konkurrenz-Namen: Namen anderer Autoren (außer bei "fans von")
- Kategorien: Nicht die exakte Kategorie-Bezeichnung
- Irreführend: Keywords die nicht zum Inhalt passen
