## Usage Tips

1. **Adapt, don't copy** - Use these as templates, modifying details for your specific needs
2. **Always include reference** - Add "Use uploaded reference photo exactly for [subject]" when using user photos
3. **Adjust technical specs** - Match lens and aperture to the scene type
4. **Maintain consistency** - Use the same style parameters across a series
5. **Layer details** - More specific details yield better results
